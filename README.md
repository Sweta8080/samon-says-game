# 🎮 Simon Says Game

## 📌 About the Project
Simon Says Game is a fun memory game built using HTML, CSS, and JavaScript. The game generates a random sequence of colors, and the player must repeat the same sequence correctly. The sequence becomes longer after each successful round, making the game more challenging.

## ✨ Features
- Interactive gameplay
- Random color sequence generation
- User click detection
- Score tracking
- Game Over message
- Responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How to Play
1. Press any key to start the game.
2. Watch the color sequence carefully.
3. Click the colors in the same order.
4. If your sequence is correct, you move to the next level.
5. If you click the wrong color, the game ends.
6. Press any key to restart the game.
